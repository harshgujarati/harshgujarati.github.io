# Harsh Gujarati - Portfolio

A clean, professional portfolio website for a Data Analyst & Business Intelligence Developer.

## About

Portfolio showcasing data analytics expertise, BI dashboard development, and technical projects. Built with vanilla HTML, CSS, and JavaScript for optimal performance.

## Features

- *Dark/Light Mode*: Toggle between themes with preference persistence
- *Responsive Design*: Mobile-first approach, works on all devices
- *Professional Summary*: Real work experience from Nation First International, Burpshy, and Devika Rubber Works
- *Technical Stack*: Comprehensive display of programming languages, BI tools, and databases
- *Project Showcase*: AtliQ Hospitality Analytics, AR Art Home Decor, House Price Prediction
- *Certifications*: Microsoft Power BI, Oracle AI certifications
- *Contact Form*: Functional form with validation

## Quick Start

1. Clone or download the repository
2. Open index.html in your browser
3. No build process or dependencies required

## Structure


├── index.html          # Main HTML file
├── styles.css          # All styling and themes
├── script.js           # Interactive functionality
└── README.md           # This file


## Customization

Update personal information in index.html:
- Contact details (email, phone, LinkedIn, GitHub)
- Work experience and achievements
- Projects and technical skills
- Education and certifications

Modify colors in styles.css by editing CSS variables in :root and [data-theme="dark"].

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter, IBM Plex Sans)
